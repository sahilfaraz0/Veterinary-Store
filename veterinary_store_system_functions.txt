================================================================================
            VETERINARY STORE & CLINIC - SYSTEM FUNCTIONS & CAPABILITIES
================================================================================
Project Name: Veterinary Store & Clinic (Islamabad, Pakistan)
Architecture: Liquid Glassmorphism UI (Vanilla CSS + Tailwind CSS) + Single-Page Application (Pure JavaScript) + Supabase Cloud Database + Local Storage Sync
User Roles  : Administrator (Owner/Manager) & Cashier (POS Operator)
================================================================================

1. CORE SYSTEM ARCHITECTURE & USER INTERFACE (UI/UX)
--------------------------------------------------------------------------------
* Liquid Glassmorphism Theme:
  - Multi-colored rotating liquid blobs embedded behind frosted glass panels, creating real-time light refraction and dynamic color gradients.
  - High-contrast typography designed specifically for both light mode and dark mode.
* Floating macOS/iOS-Style Navigation Dock:
  - Bottom-anchored pill navigation dock with responsive scaling, horizontal touch swiping (`no-scrollbar`), and pure CSS hover animations (`scale-105`).
  - Dynamic height calculation ensuring zero overlap with top header elements across all screen resolutions.
* Responsive Header Topbar:
  - Real-time alert badges for Low Stock items (`amber pulse`) and Expiring Batches (`rose pulse`).
  - Instant camera-based Barcode/QR Code scanner pop-up.
  - One-click Theme Switcher (`Light / Dark` mode toggle).
  - Secure Logout and Role Display (`Chief Administrator - ADMIN ROLE`).


2. CASHIER CAPABILITIES & DAILY OPERATIONS (POS OPERATOR)
--------------------------------------------------------------------------------
The Cashier role is optimized for high-speed checkout, customer interaction, and stock lookups without administrative clutter.

A. Point of Sale (POS Register)
  * Interactive Medicine Catalog:
    - Searchable product grid displaying medicine name, stock availability, unit price, category, and thumbnail image.
    - Click-to-add instant cart population.
  * Barcode & QR Code Camera Scanner (`html5-qrcode`):
    - Pop-up camera modal that scans physical medicine barcodes and automatically adds exact quantities to the checkout cart.
  * Live Shopping Cart / Checkout Box:
    - Dynamic quantity increments/decrements (`+` and `-`) and direct manual quantity inputs.
    - Individual item removal and clear-cart functionality.
    - Client/Pet Linking: Dropdown (`owner-select`) to link the transaction to a registered pet owner from the CRM directory.
  * Real-Time Financial Calculations:
    - Automatic Subtotal computation.
    - Tax/VAT addition calculated dynamically.
    - Customizable Discount application (`pos-discount` field) deducted before final total.
    - Grand Total calculation with instant update on item/price modification.
  * Payment Method Selection:
    - Toggle between `Cash` and `Online / Digital Transfer` (e.g., JazzCash, EasyPaisa, Bank Transfer).
  * Checkout & Invoice Generation:
    - Completes sale with instant real-time inventory deduction across local state and cloud database.
    - Generates and displays a professional, printable receipt/invoice containing store details, itemized billing, client info, and payment summary.

B. Inventory Lookups (Catalog Read Access)
  * Check live product availability, shelf locations, and retail pricing.
  * Filter inventory by categories (`Antibiotics`, `Analgesics`, `Vaccines`, `Antiparasitics`, `Supplements`).
  * View expiration dates to ensure expired batches are never sold during checkout.

C. Client Directory Lookup (CRM Read/Select)
  * Search registered pet owners by phone number or name to link sales and verify medical notes during checkout.


3. ADMINISTRATOR CAPABILITIES & EXECUTIVE MANAGEMENT (ADMIN/OWNER)
--------------------------------------------------------------------------------
The Administrator unlocks all Cashier functions plus complete Control Center access, Financial Analytics, Inventory CRUD, and Audit monitoring.

A. Dashboard (Executive Control Center)
  * Real-Time Key Performance Indicators (KPIs):
    - Today's Total Sales Revenue (`Rs.` / `PKR`).
    - Total Active Inventory Units currently in stock.
    - Low Stock Replenishment Alert card (glows when clicked from header badge).
    - Expiring Batches Alert card (glows when clicked from header badge).
  * Quick Action Shortcuts:
    - Jump to New POS Sale, Add Inventory Item, View Client Registry, and Access Reports.
  * Live Activity Feed:
    - Stream of recent store transactions with exact timestamps, payment methods, and amount totals.

B. Full Inventory Catalog Management (CRUD)
  * Add New Medicine / Veterinary Product:
    - Input Product Name, Category, SKU/Barcode, Initial Stock Quantity, Unit Price, Supplier Information, and Expiry Date.
  * Client-Side Image Compression & Upload (`browser-image-compression`):
    - Upload medicine box/bottle photos directly from mobile camera or PC; images are automatically compressed before storage.
  * Edit & Update Stock Details:
    - Modify prices, adjust inventory counts after shipments, and update batch expiry dates.
  * Delete Products:
    - Remove discontinued medicines with safety confirmation prompts.
  * Stock Status Pills:
    - Automatic categorization into `In Stock`, `Low Stock` (warning threshold), `Out of Stock`, and `Expiring Soon`.

C. Clients & Pets CRM (Full Administration)
  * Add New Pet Owners & Patients:
    - Record Owner Name, Contact Number, Address, Pet Name, Species/Breed (e.g., Cattle, Buffalo, Horse, Dog, Cat, Poultry), Age, and Medical History.
  * Update Client Medical Records:
    - Add clinical notes, ongoing prescriptions, and vaccination schedules.
  * Delete Client Profiles:
    - Clean up inactive or duplicate records.

D. Sales Tracker & Ledger Audit
  * Complete Historical Transaction Database:
    - Search sales by invoice number, client name, payment method, or date range.
  * Detailed Transaction Breakdown:
    - Inspect itemized breakdown of past receipts including unit costs, discounts given, and tax collected.
  * Receipt Reprinting & Export:
    - Re-issue duplicate invoices or export transaction histories for accounting audits.

E. Admin Reports & Business Analytics (`Chart.js`)
  * Interactive Visual Data Charts:
    - Revenue Trends Chart: Track daily/monthly financial growth.
    - Payment Method Breakdown Chart: Distinct color-coded visualization comparing `Cash` vs `Online / Digital` collections.
    - Category Performance Chart: Identify top-selling veterinary medicine categories (`Antibiotics` vs `Vaccines`, etc.).
  * Inventory Valuation Reports:
    - Calculate total financial value of current warehouse inventory.
  * Expiry Risk Analysis:
    - Generate targeted lists of batches approaching expiration for supplier return or clearance.

F. System Security & Control Center
  * Role-Based Access Control (RBAC):
    - Manage access tiers (`Admin` vs `Cashier`).
  * Audit Alerts Monitoring (`#btn-header-audit`):
    - Track critical system events, inventory deletions, and manual price overrides.
  * Cloud Database Synchronization (`@supabase/supabase-js`):
    - Seamless two-way data sync ensuring local offline continuity and cloud backup reliability.


4. SUMMARY OF EXTERNAL LIBRARIES & INTEGRATIONS
--------------------------------------------------------------------------------
* Tailwind CSS (CDN)           : Utility-first styling and custom liquid keyframe animations.
* Lucide Icons                 : Crisp vector iconography for all medical, inventory, and POS actions.
* Chart.js                     : High-performance canvas data visualization for Admin Reports.
* HTML5-QRCode                 : Web-camera based barcode and QR code scanning for fast POS entry.
* Browser Image Compression    : Automatic on-device compression for medicine photos.
* Supabase JS Client           : Real-time PostgreSQL cloud backend and authentication.
* Anime.js                     : Advanced mathematical easing and fluid transitions.
